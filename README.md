# Ennote Security Scoop Bucket

This repository is the official, automated Scoop Bucket for the [Ennote Security CLI](https://ennote.io) targeting Windows environments.

It is strictly a read-only distribution channel. The JSON manifests contained in this repository are automatically generated, cryptographically signed, and published by our secure CI/CD pipeline.

### Installation

To install the Ennote Security CLI on Windows via Scoop, execute:

```powershell
scoop bucket add ennote-io https://github.com/ennote-io/scoop-bucket
scoop install ennote
